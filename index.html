Euphoria

Live link

This full stack project was built using a React frontend, and a Python and flask backend. I chose this project because I was excited for the challenge of creating an e-commerce platform that incorporated my love for makeup. This project was truly a labor of love, throughout all the challenges that I overcame. Logged in users can browse through different products by category names. Logged in users can leave reviews on products, as well as edit and delete their reviews.

Q-Cord

Live link

This full stack project was built using a React frontend, and a Python and Flask backend. Created by a team of 4 software engineers, Q-Cord is an instant messaging social platform that is a safe place for the queer community, hence the "Q" in the name. Non-logged in users can view existing servers through the discover page. Onced signed in, users can create new servers and channels. Powered by socket.io, users can also send instant messages in channels and direct messages in real time. The biggest challege we faced was learning and integrating websockets into our backend and frontend within a weeks timeframe.

Future feature implementations

We are currently working on giving users the ability to assign roles to the members in the servers which they own.
